API

O Identidade Carioca API disponibiliza todos os serviços necessários para tratar da autenticação e autorização do usuário aos sistemas, bem como serviços para criar, alterar, envio de email de verificação e etc.

Acessando a api​

Como descrito anteriormente, o sistema deve estar devidamente cadastrado e com a role correta para ter acesso a API.

Deverá ser feita uma requisição informando o token adquirido na header da requisição HTTP.

Exemplo: 

Authorization: Bearer cn389ncoiwuencr

Clients​

O sistema deve estar cadastrado e possuir o client e secret correspondentes para acesso a API.

Entradas e Saídas​

A API consome e produz objetos do tipo JSON.

Serviços​

Os endpoints dos serviços REST do Identidade Carioca API foram documentados com a ferramenta Swagger.

Para ter acesso, clique aqui.

Consumidor e Chave Consumidor​

Em alguns serviços disponibilizados na API deve-se informar o Consumidor e Chave do Consumidor, essas credenciais são necessárias para acessar o sistema RUC que fará a consulta junto a Receita Federal do Brasil.

O cadastramento para obtenção da credencial deve ser realizado junto a equipe do RUC.

Consumindo o Serviço de Autenticação e Autorização​

Para consumo dos serviços, deve-se efetuar uma requisição HTTP para a URI do ambiente.

O retorno da requisição é um arquivo JSON, contendo todas as informações necessárias.

Exemplo em Java para consumo do Serviço​

HttpClient client = HttpClientBuilder.create().build();                

HttpGet request = new HttpGet("https://auth-idriohom.apps.rio.gov.br/v2/token/service-account");                  

request.addHeader("clientId", "sauderio");              

request.addHeader("clientPassword", "SEGREDO");              

HttpResponse response = client.execute(request);                  

InputStream inputStream = response.getEntity().getContent();                  

String responseAsString = CharStreams.toString(new InputStreamReader(inputStream, "UTF-8"));                  

System.out.println(response.getStatusLine().getStatusCode());                  

System.out.println(responseAsString);

Exemplo em C# para consumo do Serviço​

WebRequest req = WebRequest.Create(@"https://auth-idriohom.apps.rio.gov.br/v2/token/service-account");

    

                req.Method = "GET";

    

                req.Headers.Add("clientId", "sauderio");

    

                req.Headers.Add("clientPassword", "SEGREDO");

    

                HttpWebResponse resp = req.GetResponse() as HttpWebResponse;

                if (resp.StatusCode == HttpStatusCode.OK)

                {

                    using (Stream respStream = resp.GetResponseStream())

                    {

                        StreamReader reader = new StreamReader(respStream, Encoding.UTF8);

    

                        Console.WriteLine(reader.ReadToEnd());

    

                    }

                }

                else

                {

                    Console.WriteLine(string.Format("Status Code: {0}, Status Description: {1}", resp.StatusCode, resp.StatusDescription));

                }

                Console.Read();
