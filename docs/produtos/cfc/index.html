Cartão Família Carioca

O público alvo deste documento são os desenvolvedores de software da IplanRio que prestam manutenção e apoio para novas funcionalidades.

Para o desenvolvimento os seguintes recursos foram utilizados:

  • Jboss 5.2
  • JDK 1.6
  • Mysql 5.5

Configuração do Ambiente​

Container mysql

docker run --name cfcdb -v PATH_LOCAL_BD/mysql:/var/lib/mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 -d mysql:5.5 --max-allowed-packet=1073741824

Eclipse

Subversive - plugin para Eclipse que permite integrar com um repositório SVN.

subversive

Conector svn

SVN kit 1.8.7

Java Compiler - 1.6

Validation - marcar a opção Suspend all validators

Jboss 5.2

Adicionar o servidor de aplicação, definindo o caminho do servidor localmente e atribuir a runtime instalada para 1.6.

VM Arguments JBoss 5.2 Verificar valores: 

-server 
-Djava.awt.headless=true 
-Xms1024m 
-Xmx4096m 
-XX:MaxPermSize=512m 
-Dsun.rmi.dgc.client.gcInterval=3600000 
-Dsun.rmi.dgc.server.gcInterval=3600000

Configuração da aplicação​

bulid.properties:

jboss.home=PATH_SERVIDOR/jboss-eap-5.2
jboss.domain=default

cfc-dev-ds.xml - verificar credenciais atribuídas para o ambiente local.

OpenLdap.java, método verificarAutenticacaoDePessoa - para o ambiente local, alterar retorno para true. Permite logar sem autenticar no ldap.

Atenção: os arquivos acima não devem ser atualizados no repositório.

Agilizar o processamento

Devido ao tamanho da base de dados o script abaixo pode ser utilizado localmente para agilizar o processamento de uma folha, o mesmo apaga arquivos, folhas, pagamentos e creditos anteriores a 2019.

delete from ArquivoAdicionalDeDesempenho where anoCompetencia < 2019;
delete from ArquivoAdicionalDeDesempenhoCorrigido where anoCompetencia < 2019;
delete from ArquivoBeneficiarioBloqueadoIndevidamentePelaSME where anoCompetencia < 2019;
delete from ArquivoBolsaFamilia where anoCompetencia < 2019;
delete from ArquivoDeBloqueio where anoCompetencia < 2019;
delete from ArquivoDeBloqueioIndevidoPelaSMS where anoCompetencia < 2019;
delete from ArquivoDeBloqueioSMS where anoCompetencia < 2019;
delete from ArquivoDeConvertidos where anoCompetencia < 2019;
delete from ArquivoDeDesbloqueioSMS where anoCompetencia < 2019;
delete from ArquivoDeRejeicao where anoCompetencia < 2019;
delete from ArquivoDeSaque where anoCompetencia < 2019;
delete from ArquivoDeSaqueNaoEfetuado where anoCompetencia < 2019;
delete from ArquivoProJovem where anoCompetencia < 2019;
delete from ProJovem where arquivoProJovem_id < 82;
delete from BeneficiarioAdicionalDeDesempenho where anoCompetencia < 2019;
delete from BeneficiarioBolsaFamilia where anoCompetencia < 2019;
delete from ResumoFolhaDePagamento where folhaDePagamento_id < 99;
delete from FolhaDePagamento where anoCompetencia < 2019;
delete from Pagamento where anoReferenciaPagamento < 2019 and mesReferenciaPagamento <= 12;
delete from Credito where anoCorrenteCredito < 2019 and mesCorrenteCredito <= 12;

Excluir uma folha

select max(id) from FolhaDePagamento;
DELETE FROM FolhaDePagamento where id=NUMERO_DA_FOLHA;
DELETE FROM Credito where pagamento_id in 
    (select id from Pagamento where folhaDePagamento_id=NUMERO_DA_FOLHA);
DELETE FROM Pagamento where folhaDePagamento_id=NUMERO_DA_FOLHA;
DELETE FROM ArquivoResumoAnalitico where folhaDePagamento_id=NUMERO_DA_FOLHA;
DELETE FROM ResumoFolhaDePagamento where folhaDePagamento_id=NUMERO_DA_FOLHA;
